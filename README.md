

# CAVA Visualizer Desklet

A minimalist audio visualizer desklet for the Cinnamon Desktop Environment.

## AI Notice

This project was programmed entirely by AI.

## Disclaimer

Use this software at your own risk. The developer is not responsible for any mishaps, system instability, crashes, or performance issues on your machine.


## Prerequisites

This desklet requires CAVA to capture system audio. Install it via terminal:

`sudo apt update && sudo apt install cava`

## Installation

1. Move the project folder to your local desklets directory:
`~/.local/share/cinnamon/desklets/cava-visualizer@heydanielhere`
2. Restart Cinnamon (Alt + F2, type 'r', press Enter).
3. Add the visualizer from your system Desklets settings.
